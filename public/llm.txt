## App Overview
SwiftBox is a comprehensive web application that combines productivity tools and entertainment features in a single, user-friendly platform. Built with React Native and Expo, it offers a seamless cross-platform experience.

## Primary Purpose
SwiftBox serves as an all-in-one toolkit designed to enhance user productivity while providing entertaining features for leisure time. The app bridges the gap between utility and fun, making it suitable for both work and personal use.

## Core Features

### Productivity Tools
- **Image Compression**: Client-side image optimization with adjustable quality settings
- **Images to PDF**: Convert multiple images into a single PDF document
- **PDF Merger**: Combine multiple PDF files into one document
- **QR/Barcode Scanner**: Real-time scanning with camera integration

### Entertainment Features
- **2048 Game**: Classic puzzle game with scoring and level progression
- **Daily Quotes**: Inspirational quotes with sharing functionality
- **Rewards System**: Points and achievements for user engagement

### User Experience
- **Multi-language Support**: Available in English and Hindi (i18n implementation)
- **Theme Options**: Light, dark, and system-adaptive themes
- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Offline Capability**: Core features work without internet connection

## Technical Implementation
- **Framework**: React Native with Expo managed workflow
- **Language**: TypeScript for type safety
- **State Management**: React Context API
- **Storage**: AsyncStorage for user preferences and data
- **Navigation**: Expo Router with tab-based navigation
- **Styling**: Custom StyleSheet with theme-aware components
